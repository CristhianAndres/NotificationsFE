Microphone, Camera, and File Upload Permission

To allow the app to access the Microphone, Camera, and Multimedia File Upload, follow these steps:

  1. Open the Settings app on your Android device.
  2. Go to Apps or App Management.
  3. Search for and select VSnet.
  4. Tap Permissions.
  5. Enable Microphone, Camera, and File Upload.

If you need further assistance, contact our technical support.
